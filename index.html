Introduction

The code for this talk was inspired from many sources, which are listed in the references of my slides. While the initial examples come straight from Tony Hoare's book, Communicating Sequential Processes, I chose to extend the vending machine motif for pedagogical purposes; to provide minimal working examples of utilizing the fundamental primitives.
